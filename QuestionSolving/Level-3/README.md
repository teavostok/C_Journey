# Level-3 Questions

stepping up the game here. getting into functions and more complex logic.

## what's in here

- **Q1**: Calculator using functions. basic arithmetic with function prototypes.
- **Q2**: Print all the prime numbers from 2-100. this one really makes u think fr.
- **Q3**: LCM and HCF Calculator. gotta find the least common multiple and highest common factor.
- **Q4**: Power system without using pow. calculate powers the manual way.

## how to run

```bash
gcc Q1.c -o Q1
./Q1
```

same as before. pick the question, compile it, run it. nothing new here.

## notes

still no external dependencies. just pure C and gcc. if u know how to run level 1 and 2 then u got this covered already tbh.
